Blackjack em C

Este repositório contém uma implementação do jogo Blackjack desenvolvida em linguagem C para o terminal. 

AVISO: Projeto de Acessibilidade
- Este repositório serve também como a base técnica e lógica para um projeto maior. O objetivo principal desse trabalho é utilizar este código-fonte
para criar, num repositório diferente, uma versão do Blackjack totalmente adaptada e acessível para pessoas com deficiência
(para começar a ideia é só implementar acessibilidade para pessoas com problemas de visão (como daltonismo e cegueira) ou baixa visão). 
- *Link para o repositório do projeto de acessibilidade: [work in progress]*

Funcionalidades Atuais:
- Gameplay normal: Ações de pedir carta (Hit) e manter (Stand).
- Cálculo de Pontuação: Cálculo automático do valor da mão (jogador e dealer), incluindo o valor dos Ases (1 ou 11).
- Lógica da Casa (Dealer): O dealer joga de forma automática de acordo com as regras normais do casino (pedir carta até chegar a 17).
- Interface: Texto simples executado no terminal com opções: Jogar, Regras, Desativar/Ligar modo de acessibilidade
(vai ser só implementado na versão correspondente ao projeto de acessibilidade) e Sair(lol).
