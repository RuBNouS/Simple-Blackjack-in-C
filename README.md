# Blackjack em C

Este repositório contém uma implementação simples do jogo **Blackjack (21)** desenvolvida em linguagem C para execução via terminal.

Funcionalidades
* Mecânicas Originais: Ações de `Hit` (Pedir carta) e `Stand` (Parar).
* Dealer: Lógica automática onde a casa deve atingir pelo menos 17 pontos antes de parar.
* Gestão de Pontos: Cálculo de pontuação (o Ás alterna entre 1 e 11 pontos automaticamente para evitar que o jogador estoure).
* Interface : Menu interativo via terminal com validação de entradas para evitar erros de digitação.
